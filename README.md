# Store-Gamble
Gamble module for Store: Bet on amount of credits.
Credit to [Natty](https://github.com/NaathySz) for most of the code parts.

# Config
Config will be auto generated. Default:
```json
{
    "enable_gamble": true,
    "min_credits": 0,
    "max_credits": 1000,
    "gamble_commands": [
        "gamble",
        "bet",
        "g"
    ],
    "win_chance": 50,
	"cooldown": 10
}
```
